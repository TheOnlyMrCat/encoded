Commands for encoded bot

Owner commands

Only people listed in the admin string array can execute these commands

CommandDescriptionExample
-playing "game"Sets the bot's playing message to "game"-playing Something awesome
-terminateTerminates the bot's current running state, either to code or debug-terminate

Moderator commands

Only people with the Encoded Admin role in the current server can execute these commands

CommandDescriptionExample
-purge "messages"Deletes the most recent "messages" messages from the current channel's chat-purge 100
